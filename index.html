Question

Create a web application wherein the webpage plays music when the mouse pointer is still for 15 seconds.

Solution

Timer will be incremented every second to denote the idle time. Interaction with the mouse or keyboard will reset and hide the timer.

IMPORTANT NOTE:
Kindly click on screen once to interact with webpage. This is required in some browsers like Chrome as autoplay for audio/video files is no longer supported.
